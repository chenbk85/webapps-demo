关于当前示例的说明

这个示例展示了一个简单的webapp，它包含两个Page，点击这里体验一下页面的互相切换。

从哪里开始

1. 当前页面的内容在page/index/html/index.html，修改后重新`fis release`下试试

2. 查看page/index下的其它文件，修改后重新`fis release`下试试

3. 编辑router.json，新增一个`page`试试
